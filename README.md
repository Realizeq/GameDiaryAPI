# 🎮 Game Diary

Приложение для ведения личного дневника игр. Добавляй игры, отслеживай статус прохождения и ставь оценки.

## Возможности

- Добавление игр (название, платформа, статус, оценка)
- Редактирование и удаление игр
- Цветные статусы: Пройдена / В процессе / Брошена / Вишлист
- Статистика коллекции

## Технологии

- **Frontend** — WinForms
- **Backend** — ASP.NET Core Web API
- **База данных** — SQLite (реляционная)
- **Архитектура** — Монолитная

## Скачать и запустить

1. Скачать архив на [странице](https://github.com/Realizeq/GameDiary/releases/latest)
2. Распаковать в любую папку
3. Запустить `GameDiary.Frontend.exe`
